# Security Policy

## Reporting a Vulnerability

Please report security issues privately. Do not open a public issue.

Contact:
- Phone/WhatsApp: +256 700 000 000
- Response time: same day (within 24 hours)

Include:
- A clear description of the issue
- Steps to reproduce
- Impact and risk level
- Any suggested fixes

We will acknowledge receipt, investigate, and respond with next steps.
