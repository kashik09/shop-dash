# Contributing

Thanks for contributing to ShopDash.

## Setup

```bash
npm install
cp .env.example .env
npm start
```

## Development scripts

- `npm run dev` – frontend
- `npm run server` – API server
- `npm start` – both
- `npm run lint` – lint

## Guidelines

- Keep changes focused and small.
- Add or update tests when you touch logic that can break.
- Use clear commit messages that explain what and why.
- Avoid committing secrets or local env files.

## Pull requests

- Describe the problem and the fix.
- Include screenshots for UI changes.
- List any environment variables added or changed.

## Security

If you find a security issue, do not open a public issue. See `SECURITY.md`.
